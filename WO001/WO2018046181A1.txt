abstract
    Die Erfindung betrifft eine Expansionsmaschine (11), aufweisend ein Maschinengehäuse (20) mit einer drehbar gelagerten und ein Laufrad (22) tragenden Welle (21), wobei im Betrieb dem Laufrad (22) ein Medium zugeführt wird und ein eine Laufradvorderseite (26) sowie eine Laufradrückseite (27) aufweisendes Laufrad (22) in eine Drehbewegung versetzt. Erfindungsgemäß wird eine Expansionsmaschine (11) bereitgestellt, die insbesondere hinsichtlich einer Verringerung einer Leckage optimiert ist. Erreicht wird dies dadurch, dass auf der Laufradrückseite (27) ein an der Laufradrückseite (27) anliegender von einer Feder kraftbeaufschlagter Gleitring (28) anliegt. Die Feder ist als Wellfeder (29) ausgebildet und so ausgelegt, dass ein Streifenauslauf (34, 34b) der Wellfeder (29) in einem Betriebszustand einen Flächenkontakt (35a, 35b) mit einer darunter liegenden Windung (41) bildet.
