abstract
    An apparatus controls a running state of a hybrid vehicle, wherein the hybrid vehicle is configured to select, as a running mode, an EV running mode and a HV running mode, the hybrid vehicle is configured further to select an intermittent operation mode for repeating acceleration running and coasting by intermittently driving the drive wheels, in the HV running mode. The apparatus switches the EV running mode to the intermittent operation mode according to a comparison result between a power storage amount of the power storing device and a convergence value of the power storage amount in a case where the intermittent operation mode is selected, if the intermittent operation mode is required during steady running by the EV running mode.
