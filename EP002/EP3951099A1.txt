abstract
    A hydraulic machine comprises: a power source (11); an assist motor comprising an inlet port (21a) and an outlet port (21b) and assisting a torque of the power source (11); a tank (51); a collection line (29) which is connected to the inlet port (21a) and allows a fluid to flow to the inlet port; a first return line (48) which is connected to the tank (51) and allows the fluid to flow to the tank (51); a self-priming line (25) which connects the first return line (48) to the inlet port (21a) and allows the fluid to flow from the first return line (28) to the inlet port (21a); and an anti-cavitation line (55) which connects the outlet port (21b) to the self-priming line (25) and allows the fluid to flow from the outlet port (21b) to the self-priming line (25).
